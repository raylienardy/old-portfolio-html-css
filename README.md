# Portfolio Website (Early Project)

This is one of my early web development projects created during my learning phase in high school.

## 📌 Description

This project is a simple personal portfolio website built using **HTML and CSS only**.  
It was originally created during my early exploration of web development when I was first learning how to build static websites.

The website contains:

- Personal introduction
- Skills and hobbies section
- Simple social media link
- Basic sign-up and login UI (frontend only)

## 🛠️ Tech Stack

- HTML5
- CSS3

## 🎯 Purpose of This Project

This project was made as part of my learning journey in web development and early experimentation with frontend structure and layout design.

It helped me understand:

- Basic HTML structure
- CSS styling and layout
- Page organization
- Form UI design (without backend)
